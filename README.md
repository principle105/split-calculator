# Split Calculator

A tool to plan out and calculate the split on your erg pieces.

![Image](/demo.png)

## Local Development

### Setup

1. Clone the repository locally.
    ```shell
    git clone https://github.com/principle105/split-calculator
    ```
2. Install dependencies.

    ```shell
    npm i
    ```

3. Run the application.

    ```shell
    npm run dev
    ```
